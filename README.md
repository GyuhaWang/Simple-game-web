## Next Js 로 만드는 간단한 Game Web

## 개발 목표

- 기간 : 23.05.08 ~ 23.05.15 (퇴근 후 꼭 조금씩 만들자)
- Chrome 의 offline 게임에 기반을 한다. Game start, Game Over, Score system 갖추기

- ## How to start
- npm run dev

- ## 게임 설명
- 방향키 - (w,a,s,d)
- 총알 발사 - ( space bar)

  총알을 발사하여 최대한 많은 적을 없애세요!!
  적과 부딛히면 Game Over
- ## 사용 기술
- requestAnimationFrame
- contextApi
